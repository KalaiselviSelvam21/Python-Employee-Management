# ⚡ Employee Management System

A modern and beginner-friendly Employee Management System built using **Python**, **Tkinter**, and **SQLite**.  
This desktop application allows users to manage employee records with a clean dark-themed graphical interface.

---

## 🚀 Features

- ➕ Add Employee Records
- ✏️ Update Existing Employees
- 🗑️ Delete Employee Data
- 🔍 Real-Time Search Functionality
- 📋 Employee Table View
- ✅ Input Validation
- 🎨 Modern Dark UI Design
- 💾 SQLite Database Integration
- ⚡ Fast and Lightweight Desktop App

---

## 🛠️ Technologies Used

- Python 3
- Tkinter
- SQLite3
- ttk Widgets
